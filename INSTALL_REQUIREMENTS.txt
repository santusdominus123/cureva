Jika masih error OpenCV, install Visual C++ Redistributable:

Download dan install dari:
https://aka.ms/vs/17/release/vc_redist.x64.exe

Atau install semua dependencies dengan:
pip install opencv-contrib-python==4.8.1.78

Alternatif: Gunakan versi pure Tkinter (tanpa CV2 window):
python src/sam_gui_tkinter.py
